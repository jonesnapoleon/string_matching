# String Matching


## Requirements
- Python
- Pipenv

## How to run
```
pipenv shell
pipenv install
python wsgi.py
```
Then, open browser and start opening localhost:5000

> The ```pipenv install``` command is only required for the first compilation

> You can visit [this link](https://string-matching.herokuapp.com) to play with it
